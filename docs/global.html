loggerMethodConfig

Configuration for a single logging channel.

Type:
  • Object
Properties:
Name Type Attributes Default Description
type String

Either "console" or "function"
colors Boolean <optional>
 false

(Console Channel Only) Add styling to console output
name String <optional>

Name of the configuration if using multiple logging channels
showDate Boolean <optional>
 false

Display the date on output
level Integer

Only use this logging method up to this custom level
onlyLevel Integer <optional>

Log only at that level on this method of logging. overrides level property
isDebug isDebug <optional>

set the debug configuration
func function

(Custom Channel Only) Function to be invoked if using a custom plugin. ie: log to database
isRaw Boolean <optional>
 true

(Custom Channel Only) If true then an object will be returned. if false then the object will be stringified
displayFields Array <optional>
 null

Fields that should be logged. Only fields specified in the displayFields array will be logged. if empty all fields are logged.
Source:
  • LoggerInstance.js, line 233
Examples
{
     type: 'console',
     name: 'Console 1',
     colors: true,
     showDate: true,
     isDebug: {
        on: true,
        onlyLevel: 2
     }
 }
{    // Custom logging channel
     type: 'function',
     level: 3,
     func: customFunction        // customFunction = plugin
     isRaw: true,
     displayFields: ['which', 'date', 'location', 'name', 'debug', 'data'],
     isDebug: {
        on: true,
        onlyLevel: 2
     }
}
