loggerMethodConfig

Configuration for a single logging channel.

Type:
  • Object
Properties:
Name Type Attributes Default Description
type String

Either "console" or "function"
colors Boolean <optional>
 false

Add styling to console output
name String <optional>

Name of the configuration if using multiple logging channels
showDate Boolean <optional>
 false

Display the date on output
level Integer

Only use this logging method up to this custom level
onlyLevel Integer <optional>

Log only at that level on this method of logging. overrides level property
isDebug isDebug <optional>

set the debug configuration
func function

Function to be invoked if using a custom plugin. ie: log to database
