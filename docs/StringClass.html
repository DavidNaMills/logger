Methods

buildString(data, displayConfig) → {String}

Either accepts a string or an object. An array of specific fields to log. If the property is in the array then it will be added to the string. If the property in the array is not valid in the data object then "VALUE MISSING" will be used as a place holder. The order of the string depends on the array.
